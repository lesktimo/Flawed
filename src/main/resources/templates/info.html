The flaws

Here are some info about our fla.. I mean features!

Insecure Direct Object References!

When you visit your account page try changing the id. Whoops!

Sensitive Data Exposure and Missing Function Level Access Control

I believe the admin panel should be hidden and only be accessed by authorized admins. Also users' data is not encrypted. That means things like passwords are in plain text format.

Using Components with Known Vulnerabilities

This web application is running on an older version of spring boot -- 1.2.7 to be precise -- which means there are plethora of bugs. One of them is an SPEL injection with a whitelabel error page.

Broken Authentication and Session Management

If you look very closely, you can notice that certain pages show the session ID in the URI field.
